# todoApp
Todo App with vanilla JavaScript

This script is based on the following.
https://jsprimer.net/use-case/nodecli/

## Usage
on terminal:
```
$ cd js_todoApp/
$ npx @js-primer/local-server
```

then,

```
todoappのローカルサーバーを起動しました。
次のURLをブラウザで開いてください。

  URL: http://localhost:3000
```

access this URL.
